## Nukkit for Minecraft Pocket Edition 0.15.10

This project is the **original Nukkit**, designed to work exclusively with **Minecraft Pocket Edition version 0.15.10 (vanilla)**. It is focused on maintaining and using this specific version for servers and plugin development in legacy environments.

## Features
- **Original Nukkit**, unmodified  
- Works only with **Minecraft PE 0.15.10**  
- Requires **original Java 8**  
- Supports Nukkit plugin development  
- Focused on legacy Minecraft Pocket Edition versions  

## Requirements
- Java 8  
- Minecraft Pocket Edition 0.15.10 client  

## Compatibility
This project **does not support other versions** of Minecraft Pocket Edition, including earlier, later, or modified builds.  

## Project Status
Experimental and preservation project, aimed at maintaining legacy software.
